# DoorLock 게시판 제작
## 제작 기간 & 배경
* 2021-10-15 ~ 2021-12-29
* 창업 동아리 활동으로 웹 게시판을 제작하는 역할을 맞게 되었다

## Tech Stack 
* Spring Boot(MVC), Gradle, Thymeleaf, bootstrap
* AWS, RDS(MySql), Swagger

## 기능
* Create, Read, Update, Delete 
* CRUD API

## 아쉬웠던 점
인텔리제이와 스프링부트를 처음 사용하여 만들어본 프로젝트이다. 
창업 동아리 활동으로 혼자 게시판을 제작하다보니 무중단 배포에서 막히게 되었다.  
AWS 일반배포는 성공하였으나 다음에는 무중단 배포를 해보는게 목표이다.
또한 API를 처음 공부해보면서 게시판에 대한 기능 하나만 CRUD로 간단하게 만드는데서 끝났다.
다양한 API를 사용하여 게시판을 만들어보고 싶다.
